
# human-ref-input tool

create a tool (html with js) that facilitates easy human labeling of datasheet values.
on the left half of the page show an iframe with the pdf datasheet.
on the right half there is an input form.for each field generate 3 input fields and an input for test conditions.
the inputs are pre-filed with the values from the database.
when submitting the form the next datasheet is displayed.
use the parts and fields you find in datasheet_db. use normalize_condition.
pdfs are processed in random order. put the human-ref-input-tool.html in the project root.
